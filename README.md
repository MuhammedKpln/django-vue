# Quick start

# Install Python requirements
`$ git clone https://github.com/MuhammedKpln/django-vue.git && cd django-vue`\
`$ pip install -r requirements.txt`\

# Install npm requirements
`$ cd src`\
`$ npm install`\
`$ npm run dev`\

# Finish migrations
`$ python manage.py migrate`\
`$ python manage.py runserver`\
